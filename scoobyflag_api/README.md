# ScoobyFlag



## Initialisation.

Dans un premier temps pour avoir en local votre api créer votre base de donnée avec les commandes : 
``````
php bin/console d:d:c
php bin/console d:s:u -f 
``````
Ensuite pour lancer le serveur en local il faut au préalable installer wamp server pour ensuite lancer la commande suivante: 
`````` 
symfony serve
``````
